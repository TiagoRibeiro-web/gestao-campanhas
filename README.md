# 📊 Sicoob Cocred - Gestão de Campanhas

Sistema de gerenciamento de campanhas e projetos com integração com SharePoint.

## 🚀 Funcionalidades

- 📥 Sincronização com SharePoint
- 📊 Dashboard com gráficos e métricas
- ✏️ CRUD completo de campanhas
- 📈 Análise de execução orçamentária
- 🔄 Exportação para Excel

## 🛠️ Tecnologias

- React + TypeScript
- Node.js + Express
- Microsoft Graph API
- XLSX (Excel)

## 📋 Configuração

1. Clone o repositório
2. Configure o arquivo `.env` com suas credenciais
3. Execute `npm install` no frontend e backend
4. Execute `npm start` no frontend e `npm run dev` no backend
